## Base URL

`https://bendbasis.com`

All public endpoints are served under `/api/public/v1/...`.

## Authentication

Public endpoints do not require authentication.

## Response envelope

Every endpoint returns the same top-level envelope:

```json
{
  "data": {},
  "meta": null,
  "error": null
}
```

- `data`: response payload
- `meta`: optional metadata object
- `error`: `null` on success, object on failure

## Error shape

```json
{
  "data": null,
  "meta": null,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid market_id/market_ids format",
    "details": null
  }
}
```

Common `error.code` values:

- `VALIDATION_ERROR` (400)
- `INTERNAL_ERROR` (500)

## Versioning and caching

- `X-API-Version: v1` is included in responses.
- Endpoints use `Cache-Control` headers for edge caching.

Use the pages in **Public endpoints** for exact request and response formats.
