---
title: "List arbitrage opportunities"
description: "Arbitrage opportunities with normalized public fields"
---

## Endpoint

`GET /api/public/v1/funding/arbitrage`

## Query parameters

This endpoint has no query parameters.

## Request example

```bash
curl -sS "https://bendbasis.com/api/public/v1/funding/arbitrage"
```

## Response example

```json
{
  "data": {
    "rows": [
      {
        "base_asset": "XCU",
        "apr_15d": 63.87,
        "stability": 1,
        "coverage": 1,
        "long_market_id": 2396751,
        "short_market_id": 2295154
      }
    ]
  },
  "meta": null,
  "error": null
}
```

## Response fields

Each item in `data.rows` has:

- `base_asset`: `string`
- `apr_15d`: `number`
- `stability`: `number | null`
- `coverage`: `number | null`
- `long_market_id`: `number`
- `short_market_id`: `number`

## Notes

- Funding values are returned in APR.
- This endpoint intentionally does not duplicate market metadata.
- Use `/api/public/v1/funding/markets` to resolve market details by market ID.

## Cache policy

`Cache-Control: public, max-age=0, s-maxage=300, stale-while-revalidate=60`
